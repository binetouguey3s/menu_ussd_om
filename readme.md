Persistance du projet USSD Orange Money (version simplifiée)

Conception d'une application simulant un service USSD Orange MOney capable de gérer des opérations simples (achat de forfait, transfert, annulation du dernier transfert) et d' assurer la persistance des données dans un fichier, afin de conserver l'historique des opérations entre plusieurs exécutions du programme.


Environnement de travail:

- VScode (Editeur de code)
- Terminal 
- Git / Github

Lien GitHub:  https://github.com/binetouguey3s/menu_ussd_om

Fichiers créés pour ce projet:

- main.py (Fichier python )
- ussd.json (Fichier json pour enregistrer les transactions effectuées et l'etat actuel du solde grace a un dictionnaire)

"Les fonctionnalités qui ont été réalisées":

Bienvenue sur Orange Money

On a le menu principal du services USSD de Orange Money

1. Acheter un forfait 
1. Consulter le solde Orange Money
2. Effectuer des transferts
3. Acheter du crédit
4. Forfaits Internet
5. Annuler_transfert
6. Historique des transferts
7. Modifier le solde directement sur ussd.json
0. Quitter
